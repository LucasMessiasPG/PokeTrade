# Laravel 5 and Angular 2 PokeTrade.
## Angular2 RC.5

## Installation:

```
composer install
npm install
```
Create `.env` file (can be based on `.env.example`
```
php artisan key:generate
```

## Building

```
gulp
```

## Watching

```
gulp-watch
```

## Server

```
php artisan serve
```
